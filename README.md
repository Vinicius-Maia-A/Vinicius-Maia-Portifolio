## 🖥️ Sobre o Projeto

Este portfólio apresenta minha trajetória como desenvolvedor fullstack de forma criativa e imersiva, simulando uma interface de terminal de computador dos anos 80/90. O design foi pensado para transmitir minha paixão por tecnologia e código.

### ✨ Funcionalidades

- **Boot Sequence** - Animação de inicialização estilo sistema operacional
- **Navegação por seções** - Home, Skills, Projects, Experience, Contact
- **Sidebar interativa** - Navegação em árvore e barras de proficiência
- **Cards de projetos** - Com carrossel de imagens e modal detalhado
- **Design responsivo** - Adaptável para mobile e desktop
- **Tema** - Efeitos CRT, scanlines e cores neon

## 🛠️ Como Rodar

### Pré-requisitos

- Node.js (v18 ou superior)
- npm ou yarn

### Passos

```bash
# 1. Clone o repositório
git clone https://github.com/Vinicius-Maia-A/Vinicius-Maia-Portifolio.git

# 2. Instale as dependências
npm install

# 3. Inicie o servidor de desenvolvimento
npm run dev

# 4. Acesse no navegador
http://localhost:5173
```

### Comandos Disponíveis

| Comando | Descrição |
|---------|-----------|
| `npm run dev` | Inicia servidor de desenvolvimento |
| `npm run build` | Gera build de produção |
| `npm run lint` | Executa ESLint |
| `npm run preview` | Preview do build de produção |
